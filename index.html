Dự Án Nổi Bật

RAG Chatbot cho tài liệu học tập
  • Xây dựng chatbot hỏi–đáp sử dụng kỹ thuật Retrieval-Augmented Generation (RAG) hỗ trợ trả lời câu hỏi dựa trên tài liệu môn học.
  • Triển khai trích xuất dữ liệu từ PDF và lưu trữ embedding trong FAISS để truy xuất nhanh.
  • Công cụ sử dụng: Python, LangChain, FAISS, OpenAI/Ollama.
Hệ thống Phân loại tin nhắn Spam/Ham
  • Phát triển mô hình phân loại tin nhắn SMS thành spam hoặc ham kết hợp Naive Bayes với BERT để tối ưu độ chính xác.
  • Tối ưu tốc độ xử lý bằng vector hóa thủ công và tìm kiếm lân cận gần nhất với FAISS.
  • Công cụ sử dụng: Python, Pandas, NumPy, Hugging Face Transformers.
Ứng dụng Sinh mô tả ảnh và So sánh mô hình AI
  • Xây dựng ứng dụng so sánh hiệu năng giữa các kiến trúc Deep Learning: ViT-GPT2, BLIP-Large và Microsoft GIT trên cùng một giao diện.
  • Triển khai các chiến lược lấy mẫu (Sampling strategies) như Beam Search, Top-K/Top-P để điều chỉnh độ sáng tạo của nội dung.
  • Tích hợp xử lý đa phương thức (Multimodal): hỗ trợ đầu vào từ tệp tin, URL và Webcam trực tiếp kết hợp dịch tự động.
  • Công cụ sử dụng: PyTorch, Hugging Face, Streamlit, CUDA.
